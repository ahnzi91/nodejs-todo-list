# 할 일 앱

1. 할 일을 추가할 수 있다.
2. 할 일 리스트를 볼 수 있다.
3. 할 일에 대해서 끝남/안끝남 표시를 할 수 있다.
4. 할 일을 삭제할 수 있다.

## RESTful API 사용

### 주소 + http 명령어

1. 할 일을 추가할 수 있다. 👉🏻 /task post
2. 할 일 리스트를 볼 수 있다. 👉🏻 /tasks + get
3. 할 일에 대해서 끝남/안끝남 표시를 할 수 있다. 👉🏻 /tasks/:id + put
4. 할 일을 삭제할 수 있다. 👉🏻 /tasks/:id + delete
